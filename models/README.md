# Pretrained ClimateBERT v2 (demo)

**File:** `climatebert_v2_pretrained_demo.zip` (~90 KB)

Contains the logistic-regression heads trained on the v2 labeled dataset.  
The Streamlit app also downloads **ClimateBERT embeddings** from Hugging Face on first use.

## Teammate quick start

See [docs/teammate_app_setup.md](../docs/teammate_app_setup.md).

```bash
unzip models/climatebert_v2_pretrained_demo.zip -d outputs/climatebert_v2/
streamlit run app/streamlit_app.py
```
